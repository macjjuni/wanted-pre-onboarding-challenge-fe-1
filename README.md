│  │  ├─ header
│  │  └─ main
│  │
│  ├─ pages 		(📚) => 페이지 모음
│  │  ├─ detail :: Todo 상세페이지
│  │  ├─ error  :: 404 페이지
│  │  ├─ home   :: Todo 목록 페이지
│  │  ├─ join   :: 회원가입 페이지
│  │  ├─ login  :: 로그인 페이지
│  │  └─ write  :: 글작성 페이지
│  │
│  ├─ router		(🚦) => 라우팅 관련 모음
│  ├─ style.tsx (🕺) => 스타일 관련 코드
│  └─ utils

└─ yarn.lock
